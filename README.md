# Reading Materials

A curated collection of research papers, books, SOPs, and resources on aquaculture, tilapia genetics, and related topics.

## Folder Structure

| Folder | Contents |
|--------|----------|
| `01_Genetics-and-Breeding/` | GIFT development, genetic improvement programs, strain comparisons, breeding parameters, transgenic research |
| `02_Health-and-Disease/` | TiLV, TiPV, aquatic animal diseases, biosecurity, quarantine, vertical/horizontal transmission |
| `03_Hatchery-and-Culture/` | Broodstock management, hatchery operations, seed production, water quality, tank culture |
| `04_Nutrition-and-Feeding/` | Feeding management, diet formulation, feed ingredients for tilapia |
| `05_Sampling-and-Welfare/` | Non-lethal sampling, fin clipping, tagging, otoliths, anesthesia, animal welfare |
| `06_Genomics-and-Markers/` | SNP genotyping, GWAS, genomic selection, PCR, molecular methods |
| `07_SOPs-and-Protocols/` | Internal SOPs (fin clipping, sedation, tagging, hatchery), example SOPs, SOP photos, fish health procedures |
| `08_Lab-and-Biochemistry/` | Buffer recipes, gel electrophoresis, ELISA, aseptic technique, biological assays |
| `09_Books-and-Reviews/` | Textbooks, handbooks, and extended review resources on tilapia and aquaculture |
| `10_Dissemination-and-Adoption/` | GIFT adoption studies, dissemination programs, impact evaluations |
| `11_To-Read/` | Unclassified papers and items to review |

## How to Use

Browse by topic folder. Files have been renamed to descriptive titles where possible. Cryptic or unidentified filenames are held in `11_To-Read/` for manual review.
