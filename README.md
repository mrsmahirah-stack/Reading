# Reading Materials

A curated collection of research papers, books, SOPs, and resources on aquaculture, tilapia genetics, and related topics.

## File Naming Convention

All files follow the format: `Year_FirstAuthor_ShortTopic.ext`
- `Year` — 4-digit publication year, or `XXXX` if unknown
- `FirstAuthor` — first author's last name, or organization (FAO, WorldFish, etc.)
- `ShortTopic` — 2–5 word hyphenated description

## Folder Structure

| Folder | Contents |
|--------|----------|
| `01_Genetics-and-Breeding/` | GIFT development, genetic improvement, strain comparisons, sex determination, transgenic research |
| `02_Health-and-Disease/` | TiLV, TiPV, Streptococcus, biosecurity manuals, quarantine, disease transmission |
| `03_Hatchery-and-Culture/` | Broodstock management, seed production, hatchery operations, water quality, tank culture |
| `04_Nutrition-and-Feeding/` | Feeding management, diet formulation, feed ingredients for tilapia |
| `05_Sampling-and-Welfare/` | Non-lethal sampling, fin clipping, tagging, anesthesia, animal welfare |
| `06_Genomics-and-Markers/` | SNP genotyping, GWAS, genomic selection, PCR, molecular methods |
| `07_SOPs-and-Protocols/` | WorldFish internal SOPs, example SOPs, Halamid disinfection, equipment guides |
| `08_Lab-and-Biochemistry/` | Buffer recipes, gel electrophoresis, ELISA, aseptic technique, dilution guides |
| `10_Dissemination-and-Adoption/` | GIFT adoption studies, dissemination programs, impact evaluations |
